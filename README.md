# Protein Sequence Degeneracy Analysis

## Overview

This project investigates protein sequence degeneracy in myoglobin and fibronectin using ProteinMPNN and Rosetta-based protein design methods.

## Objectives
- Study protein sequence degeneracy
- Compare sequence diversity
- Analyze amino acid conservation
- compared stuctural diffrences 
- Evaluate sequence flexibility

## Tools Used

- ProteinMPNN
- Rosetta
- AlphaFold
- Chimera
- PyMOL

## Results

Multiple amino acid sequences were found to encode structurally similar proteins, demonstrating protein sequence degeneracy and evolutionary flexibility.
